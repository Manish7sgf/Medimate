## Step 1: Create Repository on GitHub

1. Go to https://github.com/new
2. **Repository name:** `medimate-pro` (or your preferred name)
3. **Description:** "AI-Powered Medical Diagnosis System with Emergency Detection"
4. **Visibility:** Public (or Private if preferred)
5. **Don't initialize** with README (we already have one)
6. Click **Create repository**

## Step 2: Add GitHub Remote & Push

Copy-paste these commands in PowerShell (in e:\medimate folder):

```powershell
# Replace YOUR_USERNAME with your GitHub username
git remote add origin https://github.com/YOUR_USERNAME/medimate-pro.git
git branch -M main
git push -u origin main
```

## Step 3: Create GitHub Token (for CLI authentication)

1. Go to https://github.com/settings/tokens/new
2. **Token name:** "MediMate Upload"
3. **Expiration:** 90 days
4. **Scopes:** Check `repo` (all)
5. Click **Generate token**
6. **Copy the token** (you won't see it again!)

## Step 4: Authenticate with Git

When prompted for password after `git push`:
- **Username:** Your GitHub username
- **Password:** Paste the token from Step 3
